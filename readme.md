# dclong/jupyterlab

Jupyter Notebook/Lab with Python 3 (many popular packages) and Python 2 (minimal packages).

## Usage 

```
docker run -d -p 8888:8888 -p 8889:8889 \
    -e DOCKER_USER=`id -un` \
    -e DOCKER_USER_ID=`id -u` \
    -e DOCKER_PASSWORD=`id -un` \
    -v $HOME:/jupyter \
    dclong/jupyterlab
```
